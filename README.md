# AI Delivery — Scrum Master Portfolio

A single-page portfolio site for **Richard J. Valeo, CSM** — Scrum Master specializing in AI/ML delivery in regulated environments.

## Overview

This page is designed as an interactive, terminal-aesthetic professional portfolio that demonstrates real-world Scrum practice across AI delivery teams. It is built as a single self-contained HTML file with no dependencies beyond Google Fonts.

## Sections

- **Hero** — Role, stats, and summary
- **Sprint Board** — Live Kanban-style sprint view
- **Metrics** — Velocity chart and sprint health data
- **AI Proof** — Evidence of AI delivery work: model validation, compliance gating, and tooling
- **Retro** — Sprint retrospective with structured format
- **Impediment Simulator** — Interactive demo of real-time unplanned work triage

## Tech

- Vanilla HTML, CSS, JavaScript — no frameworks, no build step
- IBM Plex Mono · Libre Baskerville · DM Sans (Google Fonts)
- Fully responsive; deploy anywhere static files are served

## Deployment

Works out of the box with **GitHub Pages**. Enable Pages on the `main` branch (`/ (root)`) and the site goes live at `https://<username>.github.io/<repo>/`.
